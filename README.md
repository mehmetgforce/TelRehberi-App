# TelRehberi-App
C# ve .NET kullanılarak geliştirilen bu Telefon Rehberi projesi, modern ve kullanıcı dostu bir arayüz tasarımına odaklanmaktadır. Yazılım mühendisliği eğitimim kapsamında hazırladığım bu çalışma; kişi ekleme, silme ve listeleme gibi temel fonksiyonları şık bir UI ile birleştiriyor. Temiz kod prensipleri ve sade tasarımı ön plandadır.
